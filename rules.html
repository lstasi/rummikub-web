Overview

Rummikub is a tile-based game for 2-4 players, played with 104 number tiles and 2 joker tiles. The goal is to be the first player to play all tiles from your rack.

Game Components

  • 106 tiles total:
    • 104 numbered tiles: Numbers 1-13 in four colors (2 sets each)
    • Colors: B Black, R Red, B Blue, O Orange
    • 2 joker tiles (can substitute any tile)

Initial Setup

  1. Each player draws 14 tiles randomly from the pool
  2. The remaining tiles form the "pool" for drawing
  3. Players keep their tiles hidden on their rack

Valid Combinations

Groups

Definition: 3 or 4 tiles of the same number in different colors

Examples:

  • 333 (3 in black, red, blue)
  • 7777 (7 in all four colors)

Runs

Definition: 3 or more consecutive numbers in the same color

Examples:

  • 456 (consecutive black tiles)
  • 910111213 (consecutive red tiles)

Initial Meld Requirement

  • Before placing tiles on the board, you must make an "initial meld"
  • The initial meld must have a total value of at least 30 points
  • Tile values: Numbers 1-13 are worth their face value
  • Jokers are worth the value of the tile they represent

How to Play

Your Turn Options

  1. Place tiles: Create new combinations or add to existing ones
  2. Draw a tile: If unable to play, draw one tile from the pool (ends your turn)

Playing Tiles

  • You can add tiles to existing combinations on the board
  • You can rearrange existing combinations to create new valid ones
  • All combinations on the board must remain valid after any changes
  • If you start rearranging, you must complete all changes in your turn

Joker Rules

  • Jokers can substitute for any tile in a combination
  • A joker's value is determined by its position in the combination
  • You can retrieve a joker by replacing it with the actual tile

Winning

The first player to play all tiles from their rack wins the round!

Tips for Success

  • Try to form your initial 30-point meld as early as possible
  • Look for opportunities to add single tiles to existing combinations
  • Use jokers wisely - they're valuable for completing difficult combinations
  • Pay attention to what other players are collecting
  • Don't be afraid to rearrange the board to create new opportunities

Web Game Features

  • Drag & Drop: Drag tiles from your hand to the board
  • Click Selection: Click tiles to select them, then use "Place Selected Tiles"
  • Mobile Friendly: Works great on phones and tablets
  • Real-time: Game updates automatically when other players move
  • Rule Validation: Invalid moves are prevented with helpful messages
